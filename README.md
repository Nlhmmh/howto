# HowTo

## Description

HowTo is a platform where users can browse howTos. <br/> They can also share their knowledge and make money.

## Frontend

```
Language - Dart
Framework - Flutter
Android Application
```

## Backend

```
Language - Golang
Framework - Gin

Features
1. Hot Reload (air)
2. Graceful Shutdown
3. Logging (both stdout and log file)
4. Config File
5. sqlBoiler (MySQL)
6. Error Handling
7. JWT Authentication
8. CI
9. TLS (SSL) https
```

## Database

```
MySQL
```

## Commands

### make
```
make
```
### Backend Golang Server
```
make air
```
### MySQL database
```
make db
```

## Run Frontent
```
flutter run
```
