# HowTo

## Description

HowTo is a platform where users can browse howTos. <br/> They can also share their knowledge and make money.

## Frontend

```
Language - Dart
Framework - Flutter
Android Application
```

## Backend

```
Language - Golang
Framework - Gin
```

## Database

```
MySQL
```

## Commands

### make
```
make
```
### Backend Golang Server
```
make air
```
### MySQL database
```
make db
```

## Run Frontent
```
flutter run
```
